# Airbnb Price Prediction Analysis – Salem, Oregon

## Project Overview
This project analyzes and predicts Airbnb listing prices in Salem, Oregon using machine learning techniques. The objective is to identify key factors influencing pricing and build models that support data-driven pricing decisions.

## Objectives
- Analyze listing data to understand pricing patterns
- Identify key drivers of price
- Build predictive models to estimate listing prices
- Compare model performance and select the best approach

## Dataset
- Source: Airbnb listings dataset
- Observations: 351 listings
- Features: 79 variables (property details, location, amenities)

## Tools and Technologies
- Programming: R / SAS
- Techniques: Data cleaning, feature engineering, log transformation, multicollinearity checks
- Models: LASSO Regression, Decision Tree, Random Forest

## Methodology
1. Data cleaning and preprocessing
2. Exploratory data analysis (EDA)
3. Log transformation of price to address skewness
4. Feature engineering and multicollinearity assessment
5. Model development and evaluation using RMSE

## Key Findings
- Room type, number of bedrooms, bathrooms, and accommodation capacity are primary drivers of price
- Property characteristics have greater influence than location
- Random Forest achieved the best predictive performance on the test set

## Business Impact
- Enables data-driven pricing strategies for hosts
- Supports revenue optimization
- Improves competitive positioning through better price estimation

## Future Improvements
- Expand dataset for improved generalization
- Incorporate seasonality and demand indicators
- Evaluate advanced models such as gradient boosting
